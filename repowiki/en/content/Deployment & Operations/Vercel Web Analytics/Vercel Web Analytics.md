# Vercel Web Analytics

Vercel Web Analytics provides privacy-friendly analytics for your web applications deployed on Vercel. It offers insights into visitor behavior, page performance, and user engagement without compromising user privacy.

## Overview

Vercel Web Analytics is a lightweight analytics solution designed specifically for modern web applications. It provides:

- **Privacy-First Approach**: No cookies, no personal data collection, and GDPR-compliant by default
- **Real-Time Insights**: Monitor your site's performance and visitor metrics in real-time
- **Framework Support**: Native integrations with Next.js, Remix, SvelteKit, Nuxt, Astro, and more
- **Zero Configuration**: Works out of the box once enabled in your Vercel dashboard
- **Route Tracking**: Automatic tracking of client-side routing in supported frameworks

## Key Features

### Visitor Metrics

Track essential visitor information including:
- Page views and unique visitors
- Geographic distribution of your audience
- Referrer sources and traffic channels
- Device types and browser information

### Performance Monitoring

Monitor your application's performance with:
- Page load times
- Core Web Vitals (LCP, FID, CLS)
- Real User Monitoring (RUM) data
- Performance trends over time

### Custom Events

For Pro and Enterprise plans, you can track custom events such as:
- Button clicks
- Form submissions
- Purchase completions
- Custom user interactions

## Getting Started

To start using Vercel Web Analytics:

1. Enable Web Analytics in your Vercel project dashboard
2. Install the `@vercel/analytics` package in your application
3. Integrate the Analytics component or script into your app
4. Deploy your application to Vercel

For detailed setup instructions for your specific framework, see the [Getting Started with Vercel Web Analytics](./Getting%20Started%20with%20Vercel%20Web%20Analytics.md) guide.

## Privacy & Compliance

Vercel Web Analytics is designed with privacy at its core:

- **No Cookies**: Analytics work without storing any cookies on user devices
- **No Personal Data**: No personally identifiable information (PII) is collected
- **GDPR Compliant**: Meets European data protection regulations
- **No Third-Party Tracking**: All data is processed by Vercel's infrastructure
- **Aggregated Data**: Individual user sessions cannot be reconstructed

## Pricing

Vercel Web Analytics is available on all Vercel plans:

- **Hobby Plan**: Basic analytics included
- **Pro Plan**: Advanced analytics with custom events
- **Enterprise Plan**: Full analytics suite with priority support
